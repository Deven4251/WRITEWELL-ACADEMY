## üìö Project Overview: Transforming Handwiting, Beautifully

**WRITEWELL ACADEMY** is a modern, **full-stack, animation-rich platform** dedicated to enhancing students' handwriting. We combine proven **structured techniques** with a **beautiful, performant UI** and **powerful backend processing** to deliver visible results.

### Core Value Proposition

* **Emotional Branding:** Building trust through teacher showcase and empathetic design.
* **Visible Progress:** Utilizing a unique **Before‚ÄìAfter transformation slider**.
* **Smooth UX:** Leveraging **smooth animations** and **modern UI/UX**.
* **Interactive Handling:** Secure and **interactive inquiry submission** system.

## ‚ú® Key Features & Highlights

### üé® Frontend: The Animated Experience
The UI is built to engage and delight users with superior performance and design integrity.

* **Cinematic Transitions:** Powered by **Framer Motion transitions** for extremely smooth page and component animations.
* **Interactive Storytelling:** **Word-by-word animated headings** and **Lottie animated illustrations** to capture attention.
* **Progress Showcase:** Exclusive **Before‚ÄìAfter transformation slider** to demonstrate student success instantly.
* **Branding:** Focus on **Teacher branding with emotional design**.
* **Responsive:** **Fully responsive modern UI** ensuring desktop and mobile parity.

### üóÑÔ∏è Backend: Robust & Secure API
A Node.js/Express server designed for secure, fast, and scalable inquiry handling.

* **Data Layer:** **MongoDB persistence** via Mongoose for reliable data storage.
* **Core Functionality:** Dedicated **Inquiry submission API**.
* **Security & Validation:** Implements **Validation** logic, strict **CORS** control, and **Secure environment variable configuration**.
* **Communication:** Optional email notifications via **Nodemailer** for automated communication.

### üì° API Routes: Inquiry Submission

The primary function of the backend is to securely handle contact submissions.

| Method | Endpoint | Description |
| :--- | :--- | :--- |
| **POST** | `/api/inquiry` | Submits a new parental inquiry for enrollment. |

#### Example Request Payload

```json
{
  "name": "Parent Name",
  "email": "example@gmail.com",
  "phone": "9999999999",
  "message": "I want to enroll my child."
}
```

#### Example Success Response

```json
{
  "success": true,
  "message": "Inquiry submitted successfully"
}
```
